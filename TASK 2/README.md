## Task: Linux Forensic
- Tìm hiểu chức năng một số lệnh trong Linux như ls, cd, ps, xxd, strings, grep, cat, find, file, mv, rm, echo, pwd (trình bày rõ ràng có ví dụ minh họa cụ thể, không cần liệt kê option)
- Tìm hiểu về filesystem
- Tìm hiểu các file log trong linux (ví dụ mô phỏng lại rõ ràng theo từng hành vi trong log)
